# Formula Racing 3D

Jogo de corrida estilo fórmula em 3D feito com Three.js, Vite e TypeScript.

## Comandos

```bash
npm install
npm run dev      # Desenvolvimento
npm run build    # Produção
npm run preview  # Preview da build
```

## Publicar na Vercel

1. Suba o projeto no GitHub
2. Entre na [Vercel](https://vercel.com) e importe o repositório
3. Build command: `npm run build`
4. Output directory: `dist`
5. Deploy!
